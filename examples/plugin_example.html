Welcome to Plugin-Example Home page

Sample pages and routes to call

From normal code

  • /info - Dump application module metadata from module definition (text reply)
  • /time - Current timestamp (dynamic json reply)

From plugins

  • /hello-json - Greeting (json reply)
  • /hello-text - Greeting (text reply)
  • /hello-text - Parametric Greeting, with name in path (text reply)
  • /hello-text - Parametric Greeting, with name in query arguments (text reply)
